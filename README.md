# 💰 Smart Expense Tracker

This is a simple and modern Expense Tracker website made using HTML, CSS, and JavaScript.  
The project helps users manage their daily expenses in an easy way with charts and category tracking.

## ✨ Features

- Add daily expenses
- Select expense category
- Track total spending
- View expenses in table format
- Daily expense graph
- Delete expenses
- Clean and responsive UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

## 📂 Project Files

```bash
index.html
style.css
script.js
README.md
```

## ▶️ How to Run

1. Download the project
2. Open the folder in VS Code
3. Run `index.html` using Live Server
